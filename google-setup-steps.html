Complete OAuth integration for your Travel Agency Website

1Create Google Cloud Project

First, you need to create a project in Google Cloud Console:

  1. Go to Google Cloud Console
  2. Click "Select a project" dropdown
  3. Click "New Project"
  4. Enter project name: Travel Agency Website
  5. Click "Create"
üöÄ Open Google Cloud Console

2Enable Required APIs

Enable the necessary Google APIs for authentication:

  1. In your project, go to "APIs & Services" > "Library"
  2. Search for and enable: "Google+ API"
  3. Search for and enable: "Google Identity API"
  4. Wait for APIs to be enabled (may take a few minutes)
üìö API Library

3Configure OAuth Consent Screen

Set up the consent screen that users will see when signing in:

  1. Go to "APIs & Services" > "OAuth consent screen"
  2. Choose "External" user type
  3. Fill in the required information:
App name: Travel Agency Website User support email: your-email@gmail.com Developer contact information: your-email@gmail.com
  1. In "Scopes" section, add:
- email - profile - openid
  1. Add test users (your email addresses for testing)
  2. Click "Save and Continue" through all steps
‚öôÔ∏è OAuth Consent Screen

4Create OAuth Credentials

Now create the actual OAuth 2.0 credentials:

  1. Go to "APIs & Services" > "Credentials"
  2. Click "Create Credentials" > "OAuth 2.0 Client IDs"
  3. Choose "Web application"
  4. Configure the settings:
Name: Travel Agency Web Client Authorized JavaScript origins: - http://localhost:3000 - https://yourdomain.com (add your production domain) Authorized redirect URIs: - http://localhost:3000/api/auth/google - https://yourdomain.com/api/auth/google (add your production domain)
  1. Click "Create"
  2. Copy your Client ID and Client Secret
üîë Credentials

5Configure Environment Variables

Add your Google credentials to your project:

Create/update your .env.local file:

# Google OAuth Configuration GOOGLE_CLIENT_ID=123456789-abcdefghijk.apps.googleusercontent.com GOOGLE_CLIENT_SECRET=GOCSPX-your-client-secret-here GOOGLE_REDIRECT_URI=http://localhost:3000/api/auth/google NEXT_PUBLIC_GOOGLE_CLIENT_ID=123456789-abcdefghijk.apps.googleusercontent.com # Your existing email config EMAIL_USER=trinck.official@gmail.com EMAIL_PASS=gmth hyvq vpco xwtp

Replace the example values with your actual credentials from step 4!

6Test Your Integration

Now test that everything works:

  1. Start your development server: npm run dev
  2. Go to: http://localhost:3000/auth/login
  3. Click "Continue with Google"
  4. Sign in with your Google account
  5. You should be redirected to your dashboard!
